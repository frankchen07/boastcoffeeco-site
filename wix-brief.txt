SECTION 14 — WIX-SPECIFIC IMPLEMENTATION NOTES
================================================================================

LAYOUT COMPONENTS TO USE
  Hero sections:         Wix "Strip" with 2 columns (text + image) or full-width strip
  Feature card grids:    Wix "Grid" or "Repeater" — 3 columns desktop
  Full-bleed image bg:   Wix strip with image background + color overlay on top
  Event type pills:      Wix text boxes in a flex/wrap container, or Tag element
  Split hero (text+img): Wix 2-column strip — left column has text box, right has image

SHOPIFY INTEGRATION
  The current site pulls products via the Shopify Storefront API.
  Options in Wix:
  A) Use the Wix-Shopify channel (keeps Shopify as the product source)
  B) Migrate products to Wix Stores (requires re-entering all products)
  Recommendation: Option A if you want to keep existing Shopify checkout.

CONTACT FORM
  Use Wix's built-in form builder.
  Connect submissions to: hello@boastcoffee.com

GOOGLE MAP
  Add via Wix "Google Maps" element.
  Address: MidWife & the Baker, 2320 Central Expy, Mountain View, CA 94043

FONTS
  Add both fonts via Wix's font manager before styling anything:
    Playfair Display — Google Fonts
    Inter — Google Fonts

SECTION BACKGROUNDS
  When a section needs the dark navy color (#252444), use Wix strip background color.
  When you need an image behind text, add the image as the strip background,
  then add a color overlay (use the strip's "overlay" or "tint" setting).

OPACITY ON IMAGES
  Some images are intentionally dimmed:
    Split hero images (Nitro, Espresso, Visit): 75% opacity
    About story image: 65% opacity
  Set this in Wix's image settings under "Filters" or "Opacity".
